hexo常用命令

  hexo g 是 hexo generate的简写，由此可知hexo s -> hexo server，hexo d -> hexo deploy

修改主题

  在搭建好hexo后，自带的默认主题landscape。想要符合自己风格的主题可以在官方主题下载
  下载之后将主题放在themes下面，然后在_config.yml文件中进行修改，重新部署到服务器之后生效。
